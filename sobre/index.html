O Projeto


O Quiz Bíblico iniciou como projeto pessoal, idealizado e desenvolvido no período da pandemia do Covid-19, com o objetivo de unir o aprendizado de programação ao própósito de criar uma aplicação útil. O programa proporciona a diversão de uma brincadeira saudável com amigos junto com os conhecimentos vitais da Bíblia Sagrada.

A primeira versão (1.0) do jogo foi desenvolvida em Microsoft Excel com linguagem de programação VBA e foi distribuída gratuitamente de modo limitado. A versão (2.0) foi desenvolvida para web porque nem todas as pessoas usam o Microsoft Excel devido ao custo da sua licença e para tornar possível o acesso a um público ainda maior.

Esta versão para web requereu o aprendizado da linguagem de marcação HTML5, da linguagem de estilos CSS3, da linguagem de progamação JavaScript e do Firebase, uma plataforma de serviços do Google para construção de poderosas aplicações que inclui recursos como autenticação de usuários, banco de dados em tempo real, hospedagem entre outros. Os recursos principais são:

* Configuração da partida

Para cada partida pode-se escolher a categoria, o tempo de resposta, e os pontos de cada resposta certa. A ordem das perguntas pode ser sorteada.

* Banco de Dados

As configurações personalizadas de cada usuário bem como as perguntas são armazenadas em um poderoso banco de dados realtime do Google Firebase.

* Opção de voz virtual

Pode-se jogar com a voz virtual do navegador ativada. Não é compatível com todos os navegadores. Recomendamos usar as versões mais atuais dos navegadores Google Chrome ou Edge. O programa lê a pergunta, as alternativas e inicia a contagem de tempo automaticamente. Se a voz estiver desativada, o usuário clica no botão de contagem de tempo.

* Número de jogadores

Até 10 pessoas divididas em duas equipes podem jogar. O programa chamará os participantes na sequência, alternando a equipe a cada pergunta.

* Score

Há um Score Top 10 que grava as 10 últimas melhores pontuações .
